# Space Object Classification

Space Object Classification is an AI-powered web application that identifies astrophysical objects from images using a fine-tuned MobileNetV2 deep learning model. The system classifies images into 12 categories, including planets, galaxies, asteroids, and black holes, while providing confidence scores and Grad-CAM visual explanations for model interpretability.

### Features

* Classification of 12 astrophysical object classes
* MobileNetV2-based deep learning architecture
* Flask-powered web application
* Interactive space-themed user interface
* Confidence score and class probability display
* Grad-CAM heatmap visualization for explainable AI
* Model performance metrics (Accuracy, Precision, Recall, F1 Score)

### Tech Stack

* Python
* PyTorch
* Flask
* HTML, CSS, JavaScript
* OpenCV
* Grad-CAM

This project demonstrates the integration of deep learning, explainable AI, and modern web technologies to create an interactive astronomical image classification system.
